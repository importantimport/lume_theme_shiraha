Part 1: Installation

藍+85CD calendar_month 2023-05-30

Setup Deno & Lume

As this is a Lume theme, please follow the Deno installation instructions and Lume documentation to install them first.

Install Lume Theme Shiraha

Now that you should have a basic Lume blog, modify _config.ts to use the Shiraha theme:

diff
_config.ts
1import lume from 'lume/mod.ts'
2+ import lts from 'https://deno.land/x/lume_theme_shiraha/mod.ts'
3
4- const site = lume()
5+ const site = lume().use(lts())
6
7export default site
